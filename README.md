# wechat-video-generate
___
## 微信对话视频一键生成器
### 使用方法
1. 下载本项目代码
2. 安装requirements.txt环境
3. 执行main.py文件

### 使用教程
1. 填写关键字
![](docs/1.png)
2. 填写要下载的图片数量，并点击下载
![](docs/2.png)
3. ocr会进行文字识别并展示
![](docs/3.png)
4. 手动修改ocr识别不准确的地方
![](docs/4.png)
5. 点击生成，等待生成完成
![](docs/5.png)
